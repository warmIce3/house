# Ongoing Health Trends regarding Depression and in the Virgin Islands
## This is a capstone project that analyzes a public health dataset. Trends regarding the Virgin Islands and Depression were found using Machine Models and Charts. It was found that those in the Virgin Islands are much more likely to possess diabetes, smoke, and be depressed. The connections between these illnesses can be investigated by further researching the Virgin Islands. This can affect people with these illnesses and help those in the Virgin Islands as well. Originally, there were many empty values and duplicates in the dataset. These were all fixed. I then found trends using graphs and machine learning models. However, there were a few challenges regarding this project.  It turns out, all the illnesses were at a higher amount in the Virgin Islands specifically. I would not jump to conclusions using this dataset until doing further research.  * *
